 NEXUS  Alt + key
 --------------------------------
 A / D   pane   left / right
 W / S   tab    prev / next
 1 - 5   tab    jump to tab
 --------------------------------
 Z       zoom pane
 X       close pane
 C       split left/right
 V       split top/bottom
 T       new tab
 --------------------------------
 O       open repo (launcher)
 P       pin/unpin repo as fav
 F       fuzzy open workspace
 N       name/rename workspace
 [       cycle left
 ]       cycle right
 --------------------------------
 /       toggle this pane
 --------------------------------
