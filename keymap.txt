+------------------+--------------------------------------------+
|  NEXUS KEYBINDINGS   Alt + key  (left hand)                |
+------------------+--------------------------------------------+
|  PANE NAV        |  A  <-  left pane                        |
|  1-3 horiz panes |  D  ->  right pane                       |
+------------------+--------------------------------------------+
|  TAB NAV         |  W  ^   previous tab                     |
|  1-3 tabs        |  S  v   next tab                         |
|                  |  1 2 3 4 5   jump to tab                 |
+------------------+--------------------------------------------+
|  PANE OPS        |  C   split left/right                    |
|                  |  V   split top/bottom                    |
|                  |  Z   zoom / unzoom pane                  |
|                  |  X   close pane  (confirms)              |
|                  |  T   new tab                             |
+------------------+--------------------------------------------+
|  WORKSPACES      |  F   fuzzy workspace picker              |
|                  |  R   rename current workspace            |
|                  |  [   cycle workspace left                |
|                  |  ]   cycle workspace right               |
+------------------+--------------------------------------------+
|  CLIPBOARD       |  Ctrl+Shift+C   copy                     |
|                  |  Ctrl+Shift+V   paste                    |
+------------------+--------------------------------------------+
|  HELP            |  /   open this tab  (navigate away with  |
|                  |      Alt+W/S or Alt+1-5 to return)       |
+------------------+--------------------------------------------+
